To Do List

Project Description:

The To Do List runs on JavaScript functions including eventListeners to implement a drag and drop feature, priorities, and completions. CSS is utilized to give the website a fun look and feel, as well as characterizing the task list.

Demonstrated Skills:

  • Development Languages: HTML, CSS, JavaScript
  • Tools: Visual Studio Code, GitHub
  • Hosting: Netlify @ https://todo-list-netizen.netlify.app/

Concepts Learned

  • Drag & Drop: Knowldege in CSS grew with the use of class selectors to create visual effects for the dragging feature.
  • Task Creator: My understanding of JavaScript increased as I learned how to create new elements using multiple functions.
  • localStorage: This project required me to learn about localStorage so that users may retain their tasks after closing tabs.

Challenges & Solutions:

  • The Challenge: After closing or refreshing the tab, tasks users added would be deleted.
  • The Solution: I utilized localStorage to save data which would be recalled every time the user would reopen the website.
  • The Challenge: There was no way to categorize various tasks.
  • The Solution: A prioritize button was created to automatically push important tasks to the top, but the drag feature could still override it.
  • The Challenge: Users could not efficiently repeat tasks, but had to retype it every time they needed to complete the task again.
  • The Solution: Through research, I was able to implement a restore feature, which could bring back the task. It also remembered if the task was prioritized.

Habits of Mind in Action:

  • Initiate: Jumping right into the project was key, as it helped me put my ideas into practice as soon as possible, allowing me to weed out the less important ones.
  • Reflect: To create a good final product, I had to look back and think about what other features users may wish to see.
